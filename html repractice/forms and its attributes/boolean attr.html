Boolean Attributes in HTML

Disabled Attribute



The "disabled" attribute makes an element unclickable or uneditable.

Checked Attribute

I agree to the terms

The "checked" attribute makes a checkbox checked by default.

Autoplay Attribute

The "autoplay" attribute makes the video play automatically when the page loads.

Required Attribute




The "required" attribute makes an input field mandatory to fill out before submitting the form.

Hidden Attribute

This content is hidden from view.

The "hidden" attribute hides an element from view on the page.

Readonly Attribute

The "readonly" attribute makes an input field uneditable, but the content can be selected.

Multiple Attribute

The "multiple" attribute allows the user to select more than one file.
